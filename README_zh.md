## 这是什么？

一个交互式地图 dashboard，收录了**美国（63个）**和**加拿大（46个）**全部官方认定的国家公园，共计 **109个**。每个公园都有手写的双语简介、真实游客数据，以及从维基百科实时拉取的代表照片。

整个项目是一个独立的 HTML 文件。不需要 `npm install`，不需要构建步骤，不需要后端。用浏览器打开，直接用。

---

## 功能介绍

**🗺️ 地图**
- 暗色系 Leaflet 地图，默认以北美为中心
- 🟢 绿色标记 = 美国公园 · 🔵 蓝色标记 = 加拿大公园
- 点击任意标记，地图飞速定位，底部详情面板同步弹出
- 地图边界限制在北美范围内——不会一不小心滑到南极洲
- 气泡中的公园名直接链接到对应的维基百科词条

**🔍 筛选**
- 支持中英文搜索公园名
- 按国家（🇺🇸 / 🇨🇦）和省/州二级筛选
- 年份滑块，从1872年（黄石）到2024年（Pituamkek）
- 四种排序：字母顺序、评分、建立年份、年游客量

**📋 侧边栏**
- 完整的公园排名列表，实时随筛选更新
- 鼠标悬停公园名，右侧出现 `↗` 维基百科快速跳转链接
- 一眼可见：国家徽章、所在地区、建立年份、评分、游客量
- 魁北克公园（Forillon、La Mauricie、Mingan Archipelago）显示法语原名

**📰 详情面板**
- 选中公园后从底部弹出
- 维基百科图片自动加载（REST API，4秒超时，后台预加载前18个公园）
- 统计数据：评分、年游客量、面积、建立年份、所在地
- 双语简介完全手写，不是维基百科的复制粘贴

**🌐 语言切换**
- 右上角一键中英互换
- 全面翻译：界面标签、公园名称、简介文字、面积单位（km² ↔ 万公顷）
- 英文模式下，魁北克公园显示法语名作为副标题

## 数据覆盖范围

| 国家 | 数量 | 建立年份范围 | 游客最多 |
|---|---|---|---|
| 🇺🇸 美国 | 63 | 1872–2020 | 大烟山（1290万/年） |
| 🇨🇦 加拿大 | 46 | 1885–2022 | 班夫（410万/年） |

109个公园里的几个极值：

- **最偏远**：图克图特诺盖特（西北地区）—— 有些年份全年访客不足5人
- **最大**：兰格尔-圣伊莱亚斯（阿拉斯加）—— 面积超过整个瑞士
- **最小**：门户拱门（密苏里州）—— 91英亩，主要内容是一根不锈钢拱
- **最老**：黄石，1872年 —— 地球上第一个国家公园，没有之一
- **最新**：Pituamkek（爱德华王子岛，2022年）—— 与莱诺克斯岛米克马社区联合建立
- **地质最震撼**：格罗斯莫讷（纽芬兰）—— 你可以直接走在地球地幔的岩石上

## 使用方法

用任意现代浏览器打开 `north_america_parks.html` 即可。就这样。

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git

# 打开 dashboard
open north_america_parks.html      # macOS
start north_america_parks.html     # Windows
xdg-open north_america_parks.html  # Linux
```

无需安装任何依赖。文件首次加载时从 CDN 拉取 Leaflet 和 Google Fonts，图片则在点击时按需从维基百科获取。

---

## 技术说明

- **地图**：[Leaflet.js](https://leafletjs.com/) + CartoDB 暗色瓦片
- **图片**：维基百科 REST API（`/api/rest_v1/page/summary`）—— CDN 缓存，4秒 `AbortController` 超时，后台为列表前18个公园预加载
- **数据**：所有公园坐标、统计数据、简介均为手工整理和撰写
- **体积**：单个 HTML 文件，约750行，除 CDN 脚本外零运行时依赖

## 贡献

发现坐标有偏差？某个公园的简介辜负了它的壮美？游客数据已经过时了两年？欢迎提 PR。

公园数据位于 `<script>` 代码块靠近顶部的 `PARKS` 数组中，每个条目格式如下：

```js
{
  id: 101,
  country: "CA",
  name: "Banff",
  zh: "班夫国家公园",
  region: "AB",
  lat: 51.496, lng: -115.928,
  year: 1885,
  rating: 4.9,
  visitors: 4130000,
  areakm: 6641,
  desc: "English description...",
  descZh: "中文描述...",
  nameFr: "Parc national Banff"  // 魁北克公园专用
}
```
